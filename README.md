# Clocktea

Clocktea is cool interface for time display.

Clocktea need timer supported in vim.

## Install

1. download this srcipt file

2. source Clocktea.vim

3. mv Clocktea.vim to $HOME/.vim/Clocktea.vim

## Usage

input \":Clocktea\" in command
